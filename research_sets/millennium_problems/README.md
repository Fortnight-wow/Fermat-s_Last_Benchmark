# Millennium Prize Problems

The Clay Mathematics Institute announced seven Millennium Problems in 2000.

A correct solution to any unsolved problem receives a prize of one million USD.

## Problems

1. P versus NP
2. Riemann Hypothesis
3. Yang-Mills Existence and Mass Gap
4. Navier-Stokes Existence and Smoothness
5. Birch and Swinnerton-Dyer Conjecture
6. Hodge Conjecture
7. Poincare Conjecture (Solved)

## Benchmark Usage

The repository does not attempt to solve these problems.

Instead, it extracts smaller reasoning tasks inspired by techniques appearing in related mathematical literature.

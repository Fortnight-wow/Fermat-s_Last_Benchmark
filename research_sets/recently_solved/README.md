# Recently Solved Mathematical Problems

This collection contains famous mathematical problems that remained unsolved for years or decades before receiving a proof.

These problems are useful for benchmarking AI systems because they require reasoning patterns that historically challenged the mathematical community.

## Included Topics

- Poincare Conjecture
- Kepler Conjecture
- Boolean Pythagorean Triples Problem
- Erdos Discrepancy Problem
- Sensitivity Conjecture

For each problem we provide:

1. Original statement
2. Historical context
3. Why it was difficult
4. Solution approach
5. Benchmark tasks derived from the proof

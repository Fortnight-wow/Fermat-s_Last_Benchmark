# Number Theory Challenge Set

This directory contains difficult number-theoretic problems designed to test mathematical reasoning rather than pattern matching.

## Problem Format

Each problem file contains:

- Problem ID
- Statement
- Difficulty (1-10)
- Known status
- Verification method

## Initial Challenge Problems

### FLB-NT-001
Determine whether there exist infinitely many primes p such that p + 2 is also prime.

Status: Open Problem
Difficulty: 10/10
Verification: Not available

### FLB-NT-002
Find all integer solutions to x^2 - 2y^2 = 1.

Status: Solved
Difficulty: 5/10
Verification: Symbolic
